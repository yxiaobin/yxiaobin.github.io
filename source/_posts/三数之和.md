* #### [三数之和](https://leetcode.cn/problems/3sum/)

  * 给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有和为 0 且不重复的三元组。

    注意：答案中不可以包含重复的三元组。

  * 示例

    ```
    输入：nums = [-1,0,1,2,-1,-4]
    输出：[[-1,-1,2],[-1,0,1]]
    
    
    ```

  * **List去重**

    ```java
    
    
    
    
    class Solution {
        public List<List<Integer>> threeSum(int[] a) {
            List<List<Integer>> res = new ArrayList<>();
            int n = a.length; 
            Arrays.sort(a);
            for(int i=0; i<n;i++){
                int target = - a[i];
                int j = i+1;
                int k = n-1;
                while(j<k){
                    int tmp = a[j] + a[k];
                    if(tmp>target){
                        k--;
                    }else if(tmp<target){
                        j++;
                    }else{
                        List<Integer> s = new ArrayList<>();
                        s.add(a[i]);
                        s.add(a[j]);
                        s.add(a[k]);
                        if(!res.contains(s)){ //使用list数组去重，复杂度还是挺高的
                            res.add(s);
                        }
                        j++;
                        k--;
                    }
                }
            }
            return res;
        }
    }
    ```

    

  * **数组去重**:

    class Solution {
        public List<List<Integer>> threeSum(int[] a) {
            List<List<Integer>> res = new ArrayList<>();
            int n = a.length; 
            Arrays.sort(a);
            for(int i=0; i<n;i++){
                if(i>0 && a[i] == a[i-1]){ //如果当前枚举的a 和上一个相同，则是重复的。
                    continue;
                }
                int target = - a[i];
                int j = i+1;
                int k = n-1;
                while(j<k){
                    int tmp = a[j] + a[k];
                    if(tmp>target){
                        do{
                            k--;
                        }
                        while(k>j && a[k]==a[k+1]); //去重重复的元素a[k]
                            
                    }else if(tmp<target){
                        do{
                            j++;
                        }
                        while(k>j && a[j]==a[j-1]) ; //去重重复的元素a[j]
                    }else{
                        List<Integer> s = new ArrayList<>();
                        s.add(a[i]);
                        s.add(a[j]);
                        s.add(a[k]);
                        res.add(s);
                        // if(!res.contains(s)){
                        //     res.add(s);
                        // }
                        do{
                            j++;
                        }
                        while(k>j && a[j]==a[j-1]); 
                        do{
                            k--;
                        }
                        while(k>j && a[k]==a[k+1]);
                    }
                }
            }
            return res;
        }
    }
