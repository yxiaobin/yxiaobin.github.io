T1 两数之和 题目链接

法一：暴力枚举

思路：

逐个遍历两个元素，搜索全部组合，返回满足题意的唯一解。

性能分析：

1. 时间复杂度 O（n*n）

2. 空间复杂度 O（1）

class Solution {
public int[] twoSum(int[] nums, int target) {
    // 暴力枚举
    int p = 0;
    int q = 0;
    int n = nums.length;
    for(p=0;p<n-1;p++){
        for (q=p+1; q<n;q++){
            if(nums[p] + nums[q] == target){
               return new int[]{p, q};
            }
        }
    }
    return new int[0];
    return new int[0];
}
