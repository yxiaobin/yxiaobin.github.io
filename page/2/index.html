1.

模型层一般放在 \app\http 下

创建模型语句 php artisan make:model ModelName

一般情况下，建立的模型与其表是相对应的，如果想与其他表对应，需要在模型函数里面声明

class Post extends Model
{
    //
    protected  $table = '目标表的名<!-- more -->字' ;
}

一般情况下使用都是一一对应，这样方便。
